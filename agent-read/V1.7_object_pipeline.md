# V1.7 Object Pipeline

## Scope

This document covers the RoboTwin-side object workflow for replaying FoundationPose outputs.

Boundary:
- FoundationPose extraction is upstream.
- RoboTwin consumes `poses.npz`.
- RoboTwin does not estimate FoundationPose.

- Single object replay: [run_object_pose_r1_npz.sh](/home/zaijia001/ssd/RoboTwin/code_painting/run_object_pose_r1_npz.sh)

## Input And Output

### Input

- FoundationPose `poses.npz`
- object mesh `.obj`

Typical multi-object input root:
- `/home/zaijia001/ssd/data/R1/gt_depth_vis/<task>_multi/obj_vis`

Example structure:

```text
obj_vis/
  d_pour_blue_1/
    bottle/poses.npz
    cup/poses.npz
```

### Output

- `head_cam_replay.mp4`
- optional `third_cam_replay.mp4`
- optional `head_depth_frames/`
- optional `head_anygrasp_frames/`
- `object_world_poses.npz`
- `multi_object_world_poses.npz`

## Logic

1. Read FoundationPose poses in camera frame.
2. Build a RoboTwin replay renderer.
3. Convert camera-frame object poses to RoboTwin world poses.
4. Render replay video.
5. Optionally export depth and AnyGrasp-ready RGB-D frames.

## Key Parameters

### Dataset selection

- `--input_npz`: single-object pose archive
- `--input_root`: multi-object root
- `--ids`: select only some video ids
- `--objects`: select only some object names

### Replay selection

- `--frame_start`, `--frame_end`, `--frame_stride`, `--max_frames`
- `--missing_frame_policy hide|hold_last`

### Output

- `--head_only`: head view only
- `--save_png_frames`: save RGB frames
- `--save_head_depth`: save uint16 depth PNGs
- `--save_anygrasp_frames`: save AnyGrasp-ready RGB-D

### Scene visibility

- `--hide_robot`: keep camera pose but move robot out of view
- `--disable_table`: hide table
- `--lighting_mode default|front|front_no_shadow`: lighting preset

## Why This Stage Matters

This is the bridge from FoundationPose to AnyGrasp:

- FoundationPose outputs camera-frame object poses.
- RoboTwin converts them into world-space trajectories.
- RoboTwin can then export RGB-D and intrinsics for downstream AnyGrasp inference.
