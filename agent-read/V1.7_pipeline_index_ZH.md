# V1.7 流程总览

## 这套流程包含什么

当前 `code_painting/` 这一套链路，本质上是在做：

1. 上游做人手提取
2. RoboTwin 里重放人手轨迹
3. 上游做 FoundationPose 物体位姿提取
4. RoboTwin 里重放物体轨迹
5. 读取 AnyGrasp 结果，在 RoboTwin 里挑选候选并执行

## 一句话理解

- `hand_detections_<id>.npz` -> 在 RoboTwin 里重放人手目标
- `poses.npz` -> 在 RoboTwin 里重放物体轨迹
- 重放结果导出 RGB-D -> 在别的服务器上跑 AnyGrasp
- AnyGrasp JSON + 物体世界轨迹 + 手部 NPZ -> 在 RoboTwin 里选抓取候选并执行关键帧规划

## 实际推荐顺序

1. 先拿到 `hand_detections_<id>.npz`
2. 在 RoboTwin 里确认手部 replay
3. 再拿到 FoundationPose `poses.npz`
4. 在 RoboTwin 里重放物体并导出 AnyGrasp RGB-D
5. 在别的服务器上跑 AnyGrasp
6. 回到 RoboTwin 里读取 AnyGrasp 结果做关键帧执行
